# Herrera Turismo API

This API is created to manage the version number of Herrera Turismo App (can be downloaded in [Android](https://play.google.com/store/apps/details?id=com.fduranortega.herreraturismo) and [iOS](https://apps.apple.com/us/app/herrera-turismo/id6449236660))

Can be checked [here](https://my-json-server.typicode.com/herreraturismo/herrera-turismo-api) and is implemented using **typicode json-server** ([Web](https://my-json-server.typicode.com/) and [GitHub](https://github.com/typicode/json-server))
